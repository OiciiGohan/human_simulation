・Pythonのpygameパッケージを用いて制作したシュミレーションゲームです。製作期間は2か月程度。
・障害物、通るとHPが減少する床などを含むランダムな地形がゲーム開始時に自動で生成され、人間を模した複数のエージェントがマップ上を移動し、床上にある食糧アイテムを取得したり、取得した食糧を食べることで生き延びる様子をシミュレーションするという内容です。
・エージェントにはHP、眠気、満腹度などの状態が一人一人異なる値でランダムに割り当てられ、その値によって取る行動の確率が変化するようになっています。
・エージェントの行動をある程度ランダムにしていますが、行動が完全にランダムだとエージェントがすぐに全滅してしまいシュミレーションゲームとして面白くなかったため、「壁がある方向には進まないようにする」「満腹度が低い時は持っている食糧アイテムを食べたり、食糧アイテムを拾ったり、食糧がない場所から移動したりする確率を増やす」といった工夫をしています。
